# Gerador-de-Citacoes-em-Javascript

Este projeto consiste em um Gerenciador de Citações feito atravéz de HTML5, CSS3 e JavaScript. O projeto utiliza animações, por meio da biblioteca externa GSAP, e um sistema de notificações, por meio da biblioteca externa Toastify.js, para simular requisições a um servidor.

  <img src="images/preview-gerador-de-citacoes.png" alt="Preview do Site Gerador de Citações" width="600">
</p>

## Sobre o Projeto

O objetivo deste site é melhorar minhas habilidades em Javascript. Ele foi desenvolvido para fundamentar conceitos de criação de promises para simular requisições de dados, manipulando através de Async/Await. Além disso, há a manipulação do DOM e a utilização de bibliotecas externas, utilizando GSAP para animações e Toastify.js para um sistema de notificações.

## Tecnologias

- HTML5 - Estrutura de conteúdo.
- CSS3 - Design.
- JavaScript – Lógica e interatividade.
